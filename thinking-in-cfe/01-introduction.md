# 01 · 引言

## 1.1 · 起源

1978 年,惠勒提出延迟选择思想实验 [Wheeler 1978],核心质疑波粒二象性的实在性时机:**光子在通过第一个分束器时,是否已经"决定"自己走哪条路**?如果在光子已经飞过 BS1 之后再插入或撤掉第二个分束器 BS2,观测结果仍然完美对应"是波"或"是粒子"的预言。这暗示物理实在的"内容"由测量配置决定 · 而不是由写入时机决定。

1993 年,Elitzur 与 Vaidman 提出 **干涉无相互作用测量** (Interaction-Free Measurement, IFM) [Elitzur-Vaidman 1993]:利用 Mach-Zehnder 干涉仪的两条路径,可以在某些试验中**检测到一个物体的存在,而光子从未跟物体相互作用**。著名的"未爆炸弹检测"成为科普经典。

2001 年,Mitchison 与 Jozsa 把 IFM 推广为 **反事实量子计算** [Mitchison-Jozsa 2001]:可以推断一个量子计算的结果,而该计算实际上没有运行。

2015 年,Lin 与 Lin 把这一思想形式化为 **炸弹查询复杂度** $B(f)$ [Lin-Lin 2015],并证明 $B(f) = \Theta(Q(f)^2)$ —— 其中 $Q(f)$ 是任意 Boolean 函数 $f$ 的标准量子查询复杂度。

2024-2025 年,Filatov-Auzinsh 提出 **多对象 IFM** 配置 [Filatov-Auzinsh 2024],Franco · Camillini · Galvão在通用集成光子处理器上实物实现 [Franco-Camillini-Galvão 2026]。

经过 30 余年发展,IFM/counterfactual 在物理层面已经成熟 · 但**算法学家与行业工程师几乎从未把它当作可用工具**。原因有三:

1. 物理文献写法门槛高,大部分人需要先学量子光学基础才能读
2. 没有统一的算子抽象,每个 IFM 协议都被当作"独立的物理实验"
3. 没有应用方法论,即使能读懂物理,也不知道哪些行业问题适合

## 1.2 · 本论文的贡献

本论文的目标是**抽象 + 系统化** · 不发明新物理 · 把已有的 IFM/counterfactual 工作翻译成算法学家与行业专家能直接调用的工具集。

具体贡献:

**(1) 算子抽象 (§3)**:把 IFM 物理协议抽象为参数化算子 $\Phi^{CF}_f$ · 提供算法层接口签名 · 让调用者不必懂量子光学。

**(2) 组合代数 (§4)**:定义 5 种基础组合 (串行 / 并行 / 嵌套 / 条件 / 迭代) · 推导代价合成定理草案 · 把单算子升级为可编程构件。

**(3) 3 维超越点分析 (§5)**:把 "我们超越 FT QC 吗?" 这个模糊问题拆成 D1/D2/D3 三个独立维度,明确我们在 D3 (interface domain) 上**永久独占**外部物理 oracle 的查询能力。

**(4) 减法计算范式 (§6)**:命名并定位一个跟传统加法计算正交的算法构造哲学。强调:"Subtractive" 这个词在 LLM reasoning [Tang et al. 2026] 和 counterfactual psychology [Roese 1993] 已有不同含义 · 我们使用 "**Subtractive Computation Paradigm**" 全称严格指代基于 CFE 算子的算法构造方式 · 避免歧义。

**(5) 多维复杂度分析 (§7)**:除了经典的 query complexity · 我们引入两个新维度:**disturbance complexity** ($\delta$-bounded) 与 **adversary observability complexity** (undetectability budget) · 在新的 Pareto 边界上讨论 CFE 的位置。

**(6) 6 个算法模板 (§8)**:CPA · CBB · CAL · CV · CA* · CGTS · 每个对应一类经典算法的反事实重铸。

**(7) 应用方法论 (§9)**:5 步代数化 SOP + 7 题决策树 + 5 个反模式,让领域专家不必懂量子光学即可判断 CFE 适用性。

**(8) 可挑战问题清单 (§10)**:6 个原本认为需要 FT QC 才能解决的问题 · 我们当下用 photonic IFM 即可挑战;5 个不可挑战的诚实排除。

**(9) 限界与开放问题 (§11)**:严格 falsification check + 12 个开放研究问题。

## 1.3 · 本论文不是什么

为防止误解,明确**不**做的事:

- **不发明新物理** · 所有物理基础都已存在 (Elitzur-Vaidman 1993, Mitchison-Jozsa 2001, Lin-Lin 2015, Filatov-Auzinsh 2024, Hance 2025)
- **不构造新硬件** · 当前 SOTA photonic IFM 硬件 (N ≤ 12-32) 即足够支撑 §10 的 6 个问题
- **不主张超越 FT QC 的 capability** · D1 维度上 FT QC 是 CFE 的超集 (§5)
- **不替代量子计算** · CFE 是 **specialized counterfactual co-processor** · 不是 universal QC
- **不解决 Shor / 大尺度量子化学 / HHL** · 这些是 FT QC 不可替代的领域 (§10 + §11)
