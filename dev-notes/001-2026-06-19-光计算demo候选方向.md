> **状态**:候选探索 · 未拍板 · 等 user 在 A/B/C/D 间选 或 提新方向
> **真理源**:本文件 = 本轮探索的单点真理源 · 后续 `dev-docs/` 实施计划应反向链回

## 0 · 当前基建盘点

Blender 里已经能用的原语 (3D 几何 + 动画):

- **PIC 元件库** (`addon/pic_live/builder.py`):MZI · MMI 2×2 · 微环 · 光栅耦合器
- **Wheeler 元件库** (`scripts/wheeler_demo.py`):分束器 BS · 镜 · 探测器 · 光源
- **动画**:光子沿路径走 · 关键帧 · 多光子并行
- **场景**:基板 · 摄像机 · 灯光 · 文字标签

下面 4 个候选都是"在基建上加 1-2 块乐高"就能扩展出来的方向 · 不需要从零起。

## 1 · 选型元判据

光计算 demo 最容易踩的坑:**漂亮但跟计算无关** (纯光学表演)。

挑选时优先看:

A. 能不能让人一眼看出"这是个计算机" (不是"光学奇观")
B. 工业相关性 (能不能 backing 到真实产品 / 真实论文)
C. 基建复用率 (要不要从零建新几何 / 新概念)
D. demo 故事的层次 (能不能"摆出来 → 调参数 → 看到学习/收敛"分层深入)

## 2 · 候选清单

### A · MZI 网格 = 光子矩阵乘法器 / 神经网络层 (推荐)

- **是什么**:N×N 个 MZI 拼成网格 · 每个 MZI 的相位旋钮 = 一个权重 · 光左侧输入 → 右侧输出 = `y = U · x`。Reck/Clements 定理保证 N(N-1)/2 个 MZI + 相位 = 可表示任意 N×N 酉矩阵
- **工业相关**:Lightmatter / Lightelligence / Luminous 在卖的产品 · 光子神经网络的真实硬件本体
- **demo 玩法 (3 层深入)**:
  - L1 摆出来 · 4×4 网格 · 12 个 MZI · 看几何
  - L2 调相位 · 看光从单根输入波导被"重新分配"到 4 个输出端的强度分布
  - L3 学习 · 跑梯度下降让网格学一个简单函数 (XOR / 2×2 数字识别) · 看权重收敛
- **优**:
  - 有直接的"这是计算机"故事 (跟 NVIDIA 之外的 AI 加速器对标)
  - 跟现有 MZI 基建无缝拼 (主要工作 = 排版 + 相位旋钮可视化)
  - 视觉是一整片光在网格里流 · 信息密度高
- **缺**:
  - 几何密度大 · 要小心摆位避免视觉混乱
  - L3 那层需要内置简易学习算法 (numpy / 手写梯度下降)
- **复用率**:9/10 · 主要新建 = 网格布局 helper + 相位旋钮可视化

### B · HOM dip + 玻色采样小尺寸

- **是什么**:2 个不可分辨光子从分束器两端口同时入射 · 出来必定 bunch (一起从一端出) · 绝不分开。延迟时间扫描看 dip 曲线
- **升级**:N 光子进 M×M 网格 → 输出概率分布是 #P-hard 的矩阵 permanent · 经典电脑算不动 = 量子优势 (boson sampling)
- **工业相关**:Xanadu Borealis · 中科大九章系列 · 当前 quantum advantage 主流路径之一
- **demo 玩法 (3 层深入)**:
  - L1 单 BS · 双光子入射 · 看 bunching 瞬间
  - L2 扫描延迟时间 · 绘出 dip 曲线
  - L3 N=3 进 4×4 网格 · 看输出分布并对比经典电脑算耗时
- **优**:
  - HOM 是量子光学的 "hello world" · 必备 demo
  - 玻色采样是真正的 quantum advantage 实例
  - 跟 Wheeler 衔接自然 (单光子奇观 → 双光子奇观)
- **缺**:
  - 更接近"科学奇观"而不是"通用计算"
  - L3 那层涉及 permanent 计算 · 可视化抽象
- **视觉**:扫描延迟 → dip 曲线绘出 + 2 光子在 BS 处"黏"在一起的瞬间
- **复用率**:8/10 · 主要新建 = 双光子源 + 延迟扫描 + dip 曲线绘制

### C · 量子擦除 (Wheeler 续集)

- **是什么**:在 Wheeler 的两条 path 上加偏振 tag (一红一蓝) · 原则上能知道光走了哪条 · 干涉消失;在出口加 45° 偏振片把 tag 信息**抹掉** · 干涉**又回来了** · 哪怕擦除发生在光已经"过了" BS2 之后
- **工业相关**:几乎为零 · 纯物理表演 / 教学 demo
- **demo 玩法 (3 层深入)**:
  - L1 Wheeler + 偏振 tag · 看干涉消失
  - L2 加 45° 偏振片 · 看干涉复现
  - L3 延迟擦除 · 偏振片在 BS2 后面 · 看"未来选择改变过去"
- **优**:
  - Wheeler 真正的高潮 · 大脑爆炸级
  - 跟刚做的 Wheeler 完美衔接 · 续作成本最低
- **缺**:
  - 跟"计算"几乎无关
  - 偏振是新维度 · 需要新建几何 (双折射波片)
- **视觉**:偏振滤片旋转 + 干涉条纹消失/复现
- **复用率**:9/10 · 直接在 Wheeler 上扩展

### D · 相干 Ising 机 (CIM) · 用光解 NP-hard

- **是什么**:N 个光学振荡器组成网络 · 每个有 ±1 两种相位 · 边耦合强度对应图的边权 · 网络自然演化到 Max-Cut 的最优解 (能量最低态)
- **工业相关**:NTT / Hitachi / QC Ware · 真在卖的光学组合优化器
- **demo 玩法 (3 层深入)**:
  - L1 摆出 8 节点环形拓扑 · 看初始随机相位
  - L2 注入耦合 · 看相位逐渐二值化 (±1)
  - L3 给具体图 + 已知 Max-Cut 解 · 看光学方案是否找到最优 cut
- **优**:
  - 最有"光做了 CPU 做不到的事"叙事
  - 解图问题视觉冲击强 (节点颜色随时间收敛到 2 色)
  - 跟 AI 加速 / 优化求解器商业现实接轨
- **缺**:
  - 语义离已有基建最远 · 要新建振荡器 + 二值化 + 图拓扑
  - 需要后端数值求解模拟器 (不只是几何 demo)
- **视觉**:节点环 + edge 流光 + 实时收敛到 ±1 颜色块
- **复用率**:5/10 · 大量新概念需要从零

## 3 · 私心想推但没选的 3 个

| 方向 | 一句话 | 为什么没选 |
|---|---|---|
| 光学 4f Fourier 系统 | 一片透镜瞬间做 2D 傅里叶变换 + 一片掩模做卷积 | 要画透镜光线追迹 · 不在我们 PIC 几何语言里 |
| 储备池计算 (reservoir) | 多模光纤里的混沌散斑做分类器 · 只训最后一层 | "magic" 但视觉抽象 · 难直观演示 |
| 拓扑光子学边缘态 | 光在拓扑绝缘体边缘绕障碍物不反射 | 极漂亮 · 但物理概念门槛高 · 不"友好" |

## 4 · 推荐 · A (MZI 网格)

理由按权重:

1. **基建复用率最高** · 新加只是"摆 16 个 MZI + 16 个相位旋钮" · 大量已有 MZI 元件可直接用
2. **故事性最强** · 是当前真正量产的光子算力产品 · 贴出来能立刻产生"原来 photonic AI 长这样"的解释力
3. **可分层深入** · L1 / L2 / L3 每层都自洽 · 不强制做完才有 demo · 每层结束都能截图给人看
4. **跟 Wheeler 形成对照** · Wheeler 是"光本身在算什么" · MZI 网格是"我们用光算了什么"

次推 C (量子擦除) · 因为续作成本最低 · 但 narrative 局限在物理表演。

不推 D · 复用率低 · 优先级最末。
