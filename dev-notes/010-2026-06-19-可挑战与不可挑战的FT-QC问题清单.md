# 可挑战与不可挑战的 FT QC 问题清单

> **触发**:user "是不是立即可以去挑战一些人们留给未来量子计算的问题?"
> **状态**:scope 界定 v1 · 6 个可挑战问题 + 5 个不可挑战问题 · 优先级排序完成
> **真理源**:本文件 = "我们现在用 CFE 算子能 / 不能挑战哪些原本留给 FT QC 的问题" 的单点真理源

## 0 · 起源

`dev-notes/004-009` 把算子定义 / 超越点 / 组合代数 / 范式 / 算法模板 / 代数化方法论都做完后 · 最关键的实际问题是 **"现在到底能挑哪些 FT QC-only 问题?"**。本文件给出诚实清单 + 优先级。

## 1 · 评估方法

每个候选问题问 6 个问题:

| 问 | 内容 |
|---|---|
| Q1 | 这个问题原本是不是被认为需要 FT QC 才能解? |
| Q2 | 它的本质是 query / oracle 类问题吗? (不是 unitary-heavy) |
| Q3 | 它的 oracle 能物理化吗? (R1/R2/R3 触发) |
| Q4 | CFE 算法能在当前硬件 N 范围内表达? |
| Q5 | 它的 sparsity / cost-asymmetry 假设成立吗? |
| Q6 | CFE 算法 cost 优于 / 等价 / 唯一可行 vs 经典? |

通过 Q1 + Q2 + Q3 → 候选;Q4-Q6 决定优先级。

## 2 · 6 个可挑战问题 (组 A · 我们立即能做)

### A1 · NAND-Tree / Boolean Formula 评估

**问题描述**:评估深度 $N$ 的平衡 NAND-tree (或更一般 read-once Boolean formula) · 给最终 root value。

**原 QC 方案**:Farhi-Goldstone-Gutmann 2008 quantum walk on FT QC · $O(\sqrt{N})$ 查询;Childs-Cleve-Jordan-Yonge-Mallo 2009 离散版本。

**CFE 方案**:把 tree 编为 photonic interferometer · leaf oracle 实物化 · 用 $\Phi^{CF}_{\text{NAND-tree}}$ 算根。

**Cost 对比**:CFE $B \sim O(\sqrt{N}/\delta)$ · 经典 $\Omega(N^{0.753})$ · FT QC $\sqrt{N}$ standard 查询。

**为什么 FT QC 没做**:FT QC 还没成熟 · 没人在物理硬件上跑过 NAND-tree。

**为什么我们能做**:photonic IFM 直接可实现 · 8-12 路硬件已 lab proven。

**优先级**:⭐⭐⭐⭐ · 学术影响大 · 是 quantum algorithms 教科书例题 · 我们做出 photonic 实物会被广泛 cite。

**独占性 (跟 FT QC 比)**:FT QC 来了之后这个会被替代 · 但我们有 5-10 年时间窗口。

### A2 · 单分子 / 单细胞多 assay 同时检测

**问题描述**:一个稀缺单细胞或单分子 · 想测 N 种 property · 经典每次测都破坏样本 · 一个样本只能测 3-5 次。

**原 QC 方案**:量子-enhanced microscopy + FT QC 信号处理 · 长期目标。

**CFE 方案**:N 波长 / polarization probe 并行 · 用 $\Phi^{CF}_{\text{COUNT}}$ + $\Phi^{CF}_{\text{LOC}}$ 测每个 property · 大部分 probe 不真触发 photo-bleaching。

**Cost 对比**:经典 = 一次性消耗;CFE = 同一样本测 N 个 property (N 可能 20-50)。

**为什么 FT QC 也做不到**:FT QC 必须先测样本 → 破坏 → 反事实性丢光。这是 D3 超越点。

**优先级**:⭐⭐⭐⭐⭐ · 商业市场大 (制药 / 生物医学) · FT QC 永远做不到 · 真独占。

**独占性**:永久独占。

### A3 · 光毒性敏感样本显微 + 分类

**问题描述**:活细胞 / 神经元成像 + ML 分类 · 但激光照射诱发 photo-damage · 高时空分辨成像不可能。

**原 QC 方案**:Quantum-enhanced metrology + FT QC ML 后处理 · 长期目标。

**CFE 方案**:Undetected photons imaging + 反事实多对象 IFM · 光强降 10-100×。

**Cost 对比**:经典 = 接受 photo-damage 或牺牲分辨率;CFE = 同样信息 + 1/10-1/100 光剂量。

**已有商业**:QuantIC consortium / Quantum Optical Systems 已有产品。我们的角度是把"反事实分类"加进去 · 不只是成像。

**优先级**:⭐⭐⭐⭐ · 有现成商业雏形可锚定。

**独占性**:已成熟 niche · 我们晚到 · 但分类应用还有空间。

### A4 · 物理对抗 stealth 探测 (军用 / 情报)

**问题描述**:对手部署 $N$ 个物理监控设备 (摄像头 / 监听器 / 雷达 / 物理 trap) · 想查"有没有 active device" · 但主动探测会暴露探测者。

**原 QC 方案**:Quantum radar (实验阶段 · 主要用于 stealth detection 不是 stealth probing) · FT QC 信号处理。

**CFE 方案**:$\Phi^{CF}_{\text{OR}}$ + 物理耦合 (EM / optical probe) · 反事实探测 · 对手物理上检测不到。

**Cost 对比**:经典暴露 = 损失情报优势;CFE 不暴露 = 保情报。是 capability 比较不是 cost。

**为什么 FT QC 也做不到**:同 A2 · 是 D2 超越点 (adversary-undetectable)。

**优先级**:⭐⭐⭐ · 市场小但 ARPU 极高 · 政府客户。

**独占性**:永久独占。

### A5 · 半导体晶圆 in-line 缺陷扫描

**问题描述**:wafer 上 $N$ 个 die · 想找有缺陷的 · 当前用光刻 + 电子束 inspection · 都引入 photo-damage 或 sample 损伤。

**原 QC 方案**:Quantum-enhanced metrology + FT QC ML for 缺陷分类 · 长期目标。

**CFE 方案**:多路 photonic IFM 并行扫 N 个 die · CPA 算法 · 大部分 die 不真测。

**Cost 对比**:经典 = N 次完整扫描 + 每次有 photo-damage;CFE = K << N 次真扫描 + 几乎无 damage。

**为什么 FT QC 也做不到**:同 D3 超越点 · wafer 是物理对象。

**优先级**:⭐⭐⭐⭐ · 半导体行业是大市场 · 跟 ASML 之类合作可能性。

**独占性**:永久独占。

### A6 · 稀有资源 / 核素 / 量子标记物勘探

**问题描述**:在大批样品里检测稀有元素 (核素 / 稀土) · 当前 spectroscopy 会消耗样品 · 一次性。

**原 QC 方案**:Quantum spectroscopy + FT QC 信号处理 · 长期目标。

**CFE 方案**:反事实多目标 spectroscopy · $\Phi^{CF}_{\text{COUNT}}$ 算总量 · $\Phi^{CF}_{\text{LOC}}$ 定位每个。

**Cost 对比**:经典 = 消耗大量样品;CFE = 几乎不消耗。

**优先级**:⭐⭐⭐ · 核工业 / 地质 / 稀土行业 · 客户少但客单价高。

**独占性**:永久独占。

## 3 · 5 个不可挑战问题 (组 C · 诚实排除)

### C1 · Shor 算法 / 整数分解 / RSA 攻击

**为什么不行**:Shor 不是 query 模型 · 是 modular arithmetic + QFT (quantum Fourier transform) · 跟 CFE 算子正交。Shor 必须 FT QC。

**别尝试**:任何"用 IFM 做 RSA 攻击"都是骗局。

### C2 · 大尺度量子化学模拟 (蛋白折叠 / 药物动力学)

**为什么不行**:需要长 unitary 演化模拟 Hamiltonian dynamics · 不是 query · FT QC 不可替代。CFE 顶多在最后的"找 binding site" 做反事实辅助 · 不能替代 simulation 本身。

**部分可帮助**:`A2` 单分子 IFM 检测可以验证 simulation 结果 · 但不算"挑战" simulation。

### C3 · HHL 量子线性代数 (Ax = b)

**为什么不行**:HHL 依赖 quantum phase estimation + sparse matrix oracle · 主体是 unitary 操作 · 不是 query。CFE 不适用。

### C4 · 端到端量子机器学习训练 (QNN / quantum VAE)

**为什么不行**:训练需要参数化电路深度梯度计算 · 是 unitary-heavy 而非 query-heavy。CFE 顶多在 active learning step (`A` 系列里的 CAL) 做反事实预筛 · 不能训练 QNN 本身。

### C5 · Boson sampling / 量子 supremacy benchmark

**为什么不行**:这是 sampling 模型 · 不是 query 模型。CFE 顶多做"是否有某 sampling 输出?"的反事实判定 · 不能替代 sampling 本身。

## 4 · 优先级排序 + 推荐先做

按 (独占性 × 市场 × 立即可做) 综合排序:

| 排名 | 问题 | 推荐理由 |
|---|---|---|
| 1 | **A2 · 单分子多 assay** | 永久独占 + 大市场 + 客户 narrative 强 (生物医学) + lab demo 易 · ⭐⭐⭐⭐⭐ |
| 2 | **A5 · 半导体 wafer 检测** | 永久独占 + 巨型市场 (半导体) + 工业合作机会 · ⭐⭐⭐⭐ |
| 3 | **A3 · 光毒性显微分类** | 已有商业雏形 + 客户故事易讲 + 时间窗口窄 (其他公司在做) · ⭐⭐⭐⭐ |
| 4 | **A1 · NAND-Tree 评估** | 学术影响大 + 量子算法教科书例题 + 我们做出 photonic 实物会被广泛 cite · ⭐⭐⭐⭐ |
| 5 | **A4 · 物理 stealth probing** | 永久独占 + 高 ARPU + 但 client 少 + 政府 sales cycle 长 · ⭐⭐⭐ |
| 6 | **A6 · 稀有资源勘探** | 永久独占 + niche · 客户量少 · 进入难 · ⭐⭐⭐ |

## 5 · 推荐第一步行动

如果只能选 **1 个** 立即开始 · 选 **A2 (单分子多 assay)**:

- 永久独占 (FT QC 永远做不到)
- 大市场 (生物医学 / 制药)
- 商业故事易讲 ("一次取活细胞做 50 次 assay 不破坏样本")
- 学术合作机会 (生物物理实验室都想要)
- lab demo 门槛适中 (~$500k 起步)
- 跟现有 IFM imaging 商业雏形 (QuantIC / Q O.S.) 形成 narrative 衔接

如果选 **2 个并行做** · 加 **A1 (NAND-Tree)**:

- A2 跑商业落地 + A1 跑学术影响
- A1 demo 难度更低 (纯算子表达 · 不需要生物样本)
- A1 论文容易发顶刊 · 给 A2 商业化提供 academic credibility

## 6 · 不要做的事

- 不要号称挑战 Shor / 量子化学 / HHL · 这些是 FT QC 的真主场
- 不要号称 "通用 quantum computer" · 我们是 specialized counterfactual co-processor
- 不要在 photonic N > 32 之前做大尺度 demo · 当前硬件不够
- 不要忽视 sparsity / oracle 物理化 / cost-asymmetry 三个 prerequisite (`dev-notes/009 §5` 决策树)
